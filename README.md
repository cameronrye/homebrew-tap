# Homebrew Tap

Homebrew formulae for [Clarissa](https://clarissa.run) and other tools.

## Installation

```bash
brew tap cameronrye/tap
brew install clarissa
```

Or install directly:

```bash
brew install cameronrye/tap/clarissa
```

## Available Formulae

| Formula | Description |
|---------|-------------|
| [clarissa](https://clarissa.run) | AI-powered terminal assistant with tool execution capabilities |

## Updating

```bash
brew update
brew upgrade clarissa
```

## Troubleshooting

If you encounter issues, try:

```bash
brew untap cameronrye/tap
brew tap cameronrye/tap
brew install clarissa
```
